
# DisplayName element
Specifies the name for your Office Add-in as a string up to 125 characters long.

 **Add-in type:** Content, Task pane, Mail


## Syntax:


```XML
<DisplayName DefaultValue="string ">
```

[OfficeApp](../../reference/manifest/officeapp.md)
