# Form element

UX settings for the forms that your mail add-in will use when running on a particular device (desktop, tablet, or phone).

**Add-in type:** Mail

## Syntax

```XML
<Form>
   ...
</Form>
```

## Contained in

[FormSettings](formsettings.md)

|[TabletSettings](tabletsettings.md)|
