> [!NOTE]
> The unified manifest for Microsoft 365 enables you to combine an Office Add-in with a Teams app into an App for Microsoft 365 as a single unit of development and deployment. To learn more, see [Office Add-ins with the unified manifest](../develop/unified-manifest-overview.md). For a sample of an App for Microsoft 365 that combines a Teams app and Outlook add-in, see [Discount Offers](https://github.com/OfficeDev/Microsoft-Teams-Samples/tree/main/samples/tab-add-in-combined/nodejs).
>
> We love to get your feedback about the unified manifest. If you have any suggestions, please create an issue in the repo for the [Office JavaScript Library](https://github.com/OfficeDev/office-js/issues).
